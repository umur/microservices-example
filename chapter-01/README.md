# Chapter 1: The Microservices Decision

This chapter is conceptual. It has no runnable code.

Topics covered:

- What microservices actually are (and what they are not)
- The distributed systems tax: latency, partial failure, data consistency
- Monolith vs modulith vs microservices: when each is the right choice
- Organizational drivers: Conway's Law and team topology
- The CinéTrack platform introduced as the running example

Start with Chapter 3 for the first runnable Spring Boot microservice.
